# 🏦 Banking AI System: Loan Default Prediction & Customer Segmentation

## 📌 Overview
This project builds an end-to-end AI system for the banking sector to:
- Predict loan default risk using machine learning
- Segment customers based on financial behavior
- Provide intelligent decisions for loan approval

---

## 🚀 Features
- 🔍 **Loan Default Prediction** (Supervised Learning)
- 👥 **Customer Segmentation** (K-Means Clustering)
- 📊 **Business Insights Engine**
- 🌐 **Interactive Streamlit Dashboard**

---

## 🧠 Models Used
- Logistic Regression
- Random Forest (Best Model)
- K-Means Clustering

---

## 📊 Dataset
- Credit Card Default Dataset (UCI / Kaggle)
- Includes:
  - Credit limit
  - Payment history
  - Bill statements
  - Customer demographics

---

## ⚙️ How It Works
1. Data preprocessing & feature engineering  
2. Train classification model to predict default  
3. Apply clustering to segment customers  
4. Combine both for business decision making  
5. Display results in Streamlit dashboard  

---

## 🖥️ Run the Project

### 1. Install dependencies
```bash
pip install -r requirements.txt
```
### 2. Train models
```bash
python test_classification.py
python test_clustering.py
```
### 3. Run app
```bash
streamlit run app.py
```
## 📌 Output
Default probability
Customer segment (Cluster)
Final loan decision
